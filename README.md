# 候诊安舒站网页端原型

这是“候诊安舒站：门诊候诊情绪舒缓与诊前准备微服务系统”的患者端 H5 原型。

## 功能

- 候诊流程指引
- 院内微地图示意
- 1 分钟呼吸舒缓
- 诊前准备分步表单
- 问医生问题清单
- 就诊准备小条
- 浏览器打印
- 反馈与志愿者边界说明

## 本地运行

```bash
npm install
npm run dev
```

## 构建

```bash
npm run build
```

## 隐私与边界

第一版不采集手机号、身份证号、病历号等身份信息。填写内容保存在浏览器本地，仅用于生成就诊准备小条。本工具不提供诊断、检查结果解释或用药建议。
